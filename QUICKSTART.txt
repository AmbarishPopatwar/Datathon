================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Module not found" error
Solution: Ensure you installed all requirements
    pip install -r requirements.txt

Problem: Jupyter kernel not found
Solution: Install ipykernel
    pip install ipykernel

Problem: Streamlit app won't start
Solution: Check if port 8501 is available
    streamlit run streamlit_app.py --server.port 8502

Problem: Folium map not displaying
Solution: Install streamlit-folium
    pip install streamlit-folium

Problem: Memory error in notebook
Solution: Restart kernel and run cells sequentially
